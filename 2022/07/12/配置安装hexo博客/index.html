配置安装hexo博客

1、安装node.js

https://nodejs.org/zh-cn/

https://blog.csdn.net/qq_40712862/article/details/120231621

node -v判断是否安装成功

##下载慢解决方法

  • 更换淘宝镜像下载源：npm config set registry https://registry.npm.taobao.org

  • 或国内的npm镜像命令：npm install -g cnpm --registry=https://registry.npm.taobao.org

安装指定版本

https://blog.csdn.net/weixin_44198965/article/details/124140283

https://nodejs.org/download/release/v15.7.0/

2、注册github

https://github.com/

3、安装hexo

  • 新建文件夹，打开cmd
  • npm install hexo-cli -g
  • hexo -v判断是否安装成功

卸载

  • 安装成功后可在C:\Users\Administrator\AppData\Roaming文件夹下看到npm文件夹

  • 卸载命令：卸载成功后npm文件夹会消失npm uninstall hexo-cli -g

安装指定版本

  • 查看版本hexo --version
  • 安装 Hexo 指定版本npm i hexo@4.2.1 --save

4、生成本地静态网页

  • hexo init 名称初始化
  • cd 名称进入文件夹，npm install安装依赖
  • hexo g生成静态网页
  • hexo s打开本地预览
  • hexo clean，清除缓存文件 db.json 和已生成的静态文件 public

5、上传github

  • 创建仓库create repository

  • 填写仓库名称需要和gihub账号一致，勾选Add a README file，点击创建

  • 进入仓库点击settings->Pages，找到Github Page

  • Branch分支改成main或master

6、安装git

https://git-scm.com/

配置ssh密钥

  • 打开Git Bash Here窗口，执行ssh-keygen

  • 打开此电脑–>C盘–>用户–>Administrator–>.ssh(文件夹)–>用记事本打开 id_rsa.pub 文件，全选复制

  • 打开我们的github，进入设置–>SSH公钥–>在公钥区粘贴我们所copy的公钥，点击确定输入github登陆密码进行验证即可

或者

  • 打开Git Bash Here窗口cat ~/.ssh/id_rsa.pub输出SSH Key，然后复制

克隆github项目到本地

  • git init初始化本地仓库，后会在本地文件夹下生成一个 .git 文件夹
  • git clone 仓库地址

git的推送操作

git的拉取操作

问题

如果执行git push origin master报错The project you were looking for could not be found报错，则通过(管理员运行)git config --system --unset credential.helper 清除账户信息并且在windows凭证里添加账户信息

7、连接github，git config配置:

git config --global user.name "username"
git config --global user.email "email"

  • 将username和email换成github（或者其它类似远程仓库）的用户名和邮箱

  • –global 表示全局的，即当前用户都有效，该配置会出现在 ~/.gitconfig 文件中，~表示当前用户的目录，比如：C:\Users\username\.gitconfig

  • ssh-keygen -t rsa -C "你GitHub账号的邮箱"来生成SSH Key，中途按3次回车

8、hexo 关联github

  • 打开hexo文件夹里面的**_config.yml**文件，这是hexo的配置文件修改：

  • 然后在cmd上输入npm install hexo-deployer-git --save安装GitHub推送插件

  • 安装好了以后再输入hexo g生成静态网页，再输入hexo d上传到GitHub就可以了

9、hexo 主题

https://hexo.io/themes/

安装主题npm install hexo-theme-next

10、hexo 配置

11、 新增菜单栏选项

  • 添加新页面：hexo new page “xx”
  • 在主题配置文件的menu中加上该页面
  • 在zh-CN.yml文件中加上中文意思

12、图片处理

  • 配置文件中的post_asset_folder改成true
  • npm install hexo-asset-image --save
  • 放到该文件夹下，按照md格式输入(文件名称/图片名.jpg)

13、搜索

  • npm install hexo-generator-searchdb --save
  • 站点配置文件的扩展下添加
  • 主题配置文件下，local_search改成true即可

14、新建文章

  • hexo new 文章名称

  • 存储在source/_posts下
